# Core Station: Foundational Sectors

The Core Station is the persistent foundation of the Schwarzschild Assembly. It houses the station's "Brain," "Limbs," and "Senses."

## Modules
- [aethereum-spine/](./aethereum-spine/CONTENTS.md) — The Go-based orchestration layer and master gRPC router.
- [bridge/](./bridge/CONTENTS.md) — Tactical command sector. Contains primary operational scripts.
- [machinery/](./machinery/CONTENTS.md) — The agentic manufacturing sector. Houses automated code factories.
- [protoplasm/](./protoplasm/CONTENTS.md) — The synaptic sector. Contains STASIS logic cores and safety heuristics.
- [security/](./security/CONTENTS.md) — The Safety Rail sector. Contains Rust-based formal verification machinery.
- [sensory/](./sensory/CONTENTS.md) — The observability sector. Contains OTel configurations and performance telemetry.
