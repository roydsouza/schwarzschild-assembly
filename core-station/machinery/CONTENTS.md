# Machinery: Agentic Manufacturing Factories

The Machinery sector contains the autonomous factories responsible for the generation, assurance, and analysis of spacecraft components.

## Factories
- [scaffold-engine/](./scaffold-engine/CONTENTS.md) — Responsible for mission initialization and project scaffolding.
- [code-assurance/](./code-assurance/CONTENTS.md) — Responsible for linting, security auditing, and quality gatekeeping.
- [synthetic-analyst/](./synthetic-analyst/CONTENTS.md) — Responsible for automated architectural review and policy checking.
